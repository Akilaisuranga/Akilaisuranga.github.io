- `Akila_Isuranga_CV.pdf` — **add your CV here**

## Setup your photo
In `index.html`, find this line:
```html
<div class="profile-placeholder">AK</div>
```
Replace it with:
```html
<img src="photo.jpg" alt="Akila Isuranga">
```

## Deploy on GitHub Pages
1. Create repo on GitHub (e.g. `AkilaSite`)
2. Upload all files
3. Settings → Pages → Deploy from main branch
4. Live at: `https://yourusername.github.io/AkilaSite`
